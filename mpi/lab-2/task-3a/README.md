# Lab2 Task 3a

## Сборка

```
make
```

## Запуск
### Обычная

```
./lab2.simple.out
```

### Параллельная

```
mpirun -np <N> lab2.parallel.out
```
* **N** — количество тредов

## Сохранения данных для графика
```
sh graph.sh
```
В файле `graph.csv` будет таблица вида:

| Кол-во тредов | Время работы |
|----|----|
| n | t |
